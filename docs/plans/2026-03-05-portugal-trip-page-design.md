# Portugal & Beyond — Travel Journal Page

## Concept

A single-page, mobile-first scrapbook that scrolls vertically through each day of the trip. Warm, tactile feel — torn paper edges, polaroid-style photo frames, subtle tape/pin details, and a handwritten display font. Cream/warm white background with pops of Portuguese tile blue and sunset orange.

## Layout (top to bottom)

1. **Hero** — Trip title in playful handwritten font over watercolor splash of Portuguese flag colors. Traveler names. "April 2026"
2. **Day cards** — Each day is a "journal page":
   - Date tab (torn-edge paper strip)
   - Location name in bold handwritten type
   - Polaroid-style photo(s), slightly rotated/overlapping
   - Short description of the day's plan
   - Venue names as handwritten margin notes
   - Icons: fork/knife for restaurants, binoculars for viewpoints
3. **Footer** — "See you in Portugal" handwritten sign-off

## Visual Style

- **Background:** Cream/warm white (#FFF8F0 range)
- **Accent colors:** Portuguese tile blue (#1E5FA8), sunset orange (#E8734A)
- **Font:** Handwritten display (Caveat or Kalam) + clean body (Inter or similar)
- **Photo treatment:** Polaroid frames, slight rotation, tape/pin embellishments
- **Textures:** Subtle paper grain, torn edges on date tabs

## Images

AI-generated watercolor/sketch illustrations via nanobanana MCP tool for each location. Swappable for real photos later.

## Tech Stack

- Next.js (static export) deployed to Vercel
- Tailwind CSS
- Google Fonts (Caveat + Inter)
- nanobanana for image generation
- No database needed (static content in a data file)

## What it's NOT

- No login, comments, or backend
- No complex animations or heavy JS
- No CMS — content in a simple data file
